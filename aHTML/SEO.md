# SEO入门

**爬虫出动  ===>   爬虫访问并抓取网站数据   ===>   搜索引擎收录页面  ===>   特定算法计算网站页面得分  ===>   排名。**

各大搜索引擎的爬虫名称：Google——Googlebot、百度——BaiduSpider、360——360Spider、SOSO——Sosospider、Yahoo——Yahoo! Slurp China或Yahoo!

、有道——YoudaoBot或YodaoBot、MSN——msnbot-media或msnbot、必应——bingbot（线上：compatible; bingbot/2.0）、一搜——YisouSpider、Alexa——ia   archiver、易搜——EasouSpider、即刻——JikeSpider、一淘网——EtaoSpider。

爬虫会抓取不需要登录的页面的数据返回，模拟普通用户访问网站，然后根据返回成数据进行一定的计算来去进行网站得分计算。

网络爬虫管理规则：

1. 链接长度：不要中文，长度不要太长。
2. 网站识别回访：打开速度要快，图片、视频等要少，视频几乎不会被识别，文字最容易识别。
3. 有效收录：有价值的。
4. 爬虫越多越容易被收录：链接为通道，一个页面至少有二三十个链接才算一个合格的页面。
5. 访问日志：通过访问日志观看蜘蛛的访问情况。

收录：（收录查询——搜索引擎内输入网站网址，如果搜索不到则没有被收录）（百度链接提交工具）

查看网站收录数据：[站长工具 - 站长之家 (chinaz.com)](https://tool.chinaz.com/)

权重提升：权重-用来衡量数据访问量的一个数值，每个稳定的访问量对应一个权重值。以前的Google、百度等都有权重的一个算法，后面取消了。快速提升网站权重——通过大量网站排名或追求高指数的关键字的排名

SEO核心数据计算：展现量、点击量（点击率=点击次数/展现量，站外点击（TDK）、站内点击（栏目））、访问量（跳出率=立刻离开的用户数/访问数）、停留时间（用户停留网站的时间）

关键字选择：

1. 指数：百度关键字指数里，500到600之间的指数的关键字可以选择一到两个。
2. 相关结果：百度上通过关键字搜索得出的结果应该在一千万左右的。
3. 广告位：搜索出来的广告位只有三个左右的。
4. 首页数量：在百度排名的十个自然排名网站首页中，关键字只出现在四个首页网站的。
5. 满足三个以上的关键字，可以作为关键字使用。

TDK：（占网站排名分数的40%）

1. title：网站标题。
2. meta description：网站描述。规则：字数60-80之间，抓住用户需求去写（用户的痛点）、产品在行业的优势、包含关键词。
3. meta keywords：网站关键字。关键字一般书写规则：关键字1-关键字2-关键字3-品牌词。

栏目：用于引导用户点击

1. 栏目数量：6-9个。
2. 切合用户需求、产品的特点。
3. 栏目的顺序：越重要的位置越靠前。

锚文本——超链接文本：点击可以跳转到另一个页面的文字

1. 作用：增加点击率、增加用户留存时间、蜘蛛存留率（每一个链接都是蜘蛛的一个入口）。
2. 设置要求：
   1. 文字要和链接内容相匹配；
   2. 文字最好是要排名的关键字；
   3. 设置密度不能超过1%，一般500字文章不超过两个，1000字不超过四个。
   4. 不能有多个链接都指向一篇文章（因为会造成大量回链，百度会检测到）。
   5. 站在用户需求角度设置。（如果锚文本点击率很低甚至没有，那就会起反作用，拖累排名；即使一篇文章中设置了多个锚文本，那么如果点击效果很好，那也是可以的）

网站文章写作技巧：

1. 图文结合，图片1-3张左右。
2. 字数要求，八百字左右。
3. 不要乱写。

SEO外链算法：

1. 外链，站外链接。
2. 百度绿萝算法：检测优质外链。
3. 友情链接。

闪电算法——秒排名的数据要求：

1. 闪电算法：百度针对网站打开速度的一项优化规则。（网站加载速度要小于1s）
2. 影响网站数据的相关文件：html/css/js、图片、服务器速度。
3. 根据浏览器的开发者工具查看网站加载过程中哪一些文件加载速度慢从而进行针对性的优化。

第一印象规则：网站SEO优化好之后再上线。

a标签的rel属性值——nofollow：

1. 外部链接加上。
2. 广告链接加上。
3. 不重要的页面的链接加上。

图片的alt优化：

1. 每一张图片都要有alt。
2. alt描述时要有需要排名的关键字。
